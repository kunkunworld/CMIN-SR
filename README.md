# CMIN-SR

**Validity-aware spectral diagnostics for multifractal random walk inference
under finite samples.**

CMIN-SR is a Python implementation of a finite-sample spectral diagnostic
framework for stochastic time series. It studies whether an empirical scaling
spectrum is better interpreted as linear monofractal, low-curvature boundary,
or compatible with a multifractal random walk (MRW) projection family.

The framework is deliberately conservative: it organizes spectral evidence and
diagnostic warnings. It is not a proof that a signal was generated by an MRW
mechanism, and `lambda2_proj` should be interpreted as a projection coordinate,
not as a guaranteed estimate of a true intermittency parameter.

## Method Overview

```text
finite signal
  -> empirical scaling spectrum zeta(q)
  -> monofractal projection
  -> MRW-family projection
  -> residual, curvature, boundary, and instability diagnostics
  -> validity-aware diagnostic scores
```

The main diagnostic quantities are:

- `p_scaling`: evidence for stable scaling over the tested scales;
- `p_curved`: evidence for nonlinear spectral curvature;
- `p_mono`: compatibility with a linear monofractal explanation;
- `p_MRW`: compatibility with the MRW projection family;
- `p_boundary`: evidence for a low-curvature monofractal/MRW boundary case.

## Installation

Install the basic dependencies:

```bash
pip install -r requirements.txt
```

The code was developed in a local scientific Python environment with PyTorch,
NumPy, pandas, and matplotlib.

## Main Components

- **Empirical spectrum estimation:** structure-function based estimation of
  `zeta(q)` from finite increments.
- **Projection geometry:** competing monofractal and MRW-family projections.
- **Curvature diagnostics:** deterministic features that summarize curvature,
  linearity, MRW-vs-mono gain, and boundary behavior.
- **Spectral geometry calibrator:** a small spectrum-space model that maps
  spectral features to diagnostic scores.
- **Finite-sample identifiability study:** deterministic and classical
  estimators for testing whether MRW curvature can be recovered under limited
  sample length, scale range, and q-grid.
