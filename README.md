# CMIN-SR

**Validity-aware spectral diagnostics for multifractal random walk inference
under finite samples.**

CMIN-SR is a diagnostic framework for studying whether a finite stochastic
signal contains stable spectral evidence for a linear monofractal,
low-curvature boundary, or multifractal-random-walk-like interpretation. The
project is motivated by a practical difficulty in multifractal analysis:
finite windows, limited scale ranges, unstable high-order moments, and
surrogate process effects can create apparent curvature in empirical scaling
spectra.

The framework should be read as a validity-aware spectral diagnostic pipeline,
not as a guaranteed short-window recovery engine for the true MRW intermittency
parameter.

## Core Idea

```text
finite signal
  -> empirical scaling spectrum zeta(q)
  -> monofractal and MRW-family projections
  -> residual, curvature, boundary, and instability diagnostics
  -> validity-aware diagnostic scores
```

The main diagnostic quantities are:

- `p_scaling`: evidence for stable scaling over the tested scales;
- `p_curved`: evidence for nonlinear spectral curvature;
- `p_mono`: compatibility with a linear monofractal explanation;
- `p_MRW`: compatibility with the MRW projection family;
- `p_boundary`: evidence for a low-curvature monofractal/MRW boundary case.

The projected coordinate `lambda2_proj` is a coordinate within the selected MRW
projection family. It is not proof that the observed signal was generated by an
MRW mechanism.

## What Is Included

- `src/mrw_inverse/`: core models, projections, datasets, losses, and analysis
  modules.
- `experiments/`: training, evaluation, ablation, baseline, and identifiability
  scripts.
- `experiments/paper/`: paper-level wrappers and figure/table asset generation.
- `scripts/`: smoke tests and utility scripts.
- `docs/`: public documentation for the framework and main experiments.
- `paper_assets/`: paper-ready figures, tables, summaries, and figure data.
- `paper_writing_workspace/`: LaTeX manuscript source, bibliography, figures,
  and compiled draft PDF.

Large checkpoints, raw data, and full generated output folders are intentionally
not included in this public upload package.

## Installation

Create a Python environment with PyTorch, NumPy, pandas, matplotlib, and the
standard scientific Python stack. A minimal dependency list is provided in
`requirements.txt`; depending on which experiment scripts are used, additional
packages from your local scientific Python distribution may be needed.

```bash
pip install -r requirements.txt
```

The development environment used during the project was a local Conda
environment, but the code is ordinary Python and does not require that exact
environment name.

`experiments/`, but the main scientific conclusion does not require retraining
every historical checkpoint.

## Main Findings

- Analytic spectral geometry is learnable: controlled linear, boundary, and
  curved spectra are separable in clean `zeta(q)` space.
- Applying this interpretation to raw finite samples exposes an upstream
  bottleneck: empirical spectrum estimation.
- Raw zeta alignment can suppress false curvature in monofractal controls, but
  may also flatten genuine MRW curvature.
- Deterministic and classical finite-sample estimators, including structure
  functions, MFDFA, and lightweight wavelet-style controls, do not fully remove
  the short-window curvature recovery bottleneck in the tested regime.
- CMIN-SR is therefore best interpreted as a diagnostic framework for organizing
  multifractal spectral evidence, not as a mechanism validator.
