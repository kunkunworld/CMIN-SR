# Documentation Guide

This directory keeps public-facing documentation for CMIN-SR.

Recommended reading order:

1. `spectral_representation_framework.md`
2. `core_code_index.md`
3. `experiment_plan.md`
4. `finite_sample_curvature_identifiability.md`
5. `estimator_curvature_recovery.md`
6. `classical_multifractal_baselines.md`
7. `cmin_sr_failure_attribution.md`
8. `spectrum_space_calibration.md`
9. `raw_zeta_alignment.md`
10. `curvature_preserving_zeta_alignment.md`

Historical ablation notes are kept only when they help reproduce or interpret
the final diagnostic framework. Internal review notes, TODO audits, and
development logs are intentionally excluded from the public upload bundle.
